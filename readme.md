## Getting Started 

### Download and run:  
  
* `npm install` 
* `npm run start` (for start)
* `npm run dev` (for start with hot reload)

### You can try:

* `localhost:3000`
* `localhost:3000/info`
* `localhost:3000/users`
* `localhost:3000/cards`
* `localhost:3000/users/8340d0ec33270a25f2413b69`
* `localhost:3000/users/f20c9c560aa652a72cba323f`
