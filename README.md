## Getting Started 

### Download and run:  
  
`git clone https://github.com/koshelev-valerii/MestoBack`

`npm install` 
`npm run start` (for start)
`npm run dev` (for start with hot reload)

## API Resource "Cards"




## Authentication

### Signup user

`POST /signup`

### Signin user

`POST /signin`

#### Get card list

`GET /cards`

#### Create new card

`POST /cards`

#### Delete card by id

`DELETE /cards/:id`

## API Resource "Users"



#### Get user list

`GET /users`

#### Get user by id

`GET /users/:id`

#### Create new user

`POST /users`
